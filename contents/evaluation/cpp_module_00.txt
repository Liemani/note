# subject pdf

- 모든 output 은 standard output 으로, 그리고 다른 언급이 없다면 newline 으로 끝나야 한다
- 파일 이름과 class 이름, function 이름, method 이름은 letter 다음에 와야 한다.
- 사용 불가능: *alloc, *printf, free
- standard library 에 있는 것은 다 쓸 수 있다. 그러나 새로운 언어를 시작하는 만큼 c 보다는 cpp 스러운 함수를 사용하는 것이 현명할 것이다.
- STL 은 module 08 부터 가능하다
  - It means no vectors/lists/maps/etc... <algorithm> also
- 금지된 함수와 mechanic 은 0 점
- "unsing namespace", "friend" are forbidden
- 다른 요구 사항이 없다면 class 와 관련된 파일들은 항상 ClassName.hpp 와 ClassName.cpp 이어야 한다
- 제출할 directory name is ex00/, ex01/, ..., exn/
- you are not allowed to use any external library
  - also means no C++11 and derivates, nor Boost or anything
- 컴파일: 'clang++ -Wall -Wextra -Werror'
- include 는 독립적으로 직접 각각을 해줘야 한다
- 평가자가 읽을 수 없는 코드는 점수를 받을 수 없다
- 양식에 맞지 않으면 기계 평가에서 점수를 받을 수 없다
  - 놓친 것은 없는지 꼼꼼이 확인해보라



  - Namespace, class, member functions, stdio stream, initialization lists, static, const, and lots of basic stuff
